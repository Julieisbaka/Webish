# Tools Directory

This directory contains all the tools distributed through the Webish platform.

## Structure

- Each tool should be in its own subdirectory
- Include a README.md file for each tool with usage instructions
- Version control through Git tags

## Guidelines

1. Tool names should be descriptive
2. Include version information
3. Provide clear documentation
4. Test before uploading

## Security

- All tools are scanned for malware
- Code reviews are required
- Sign your commits
